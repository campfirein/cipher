# ByteRover CLI Command Reference

## Memory Commands

### `br add`

**Description:** Add or update a bullet in the playbook (bypasses ACE workflow for direct agent usage)

### `br mem retrieve`

**Description:** Retrieve memories from ByteRover Memora service and save to local playbook

### `br mem push`

**Description:** Push playbook to ByteRover memory storage and clean up local ACE files

### `br show`

**Description:** Display local context managed by ByteRover CLI

### `br ace stats`

**Description:** Display ACE playbook statistics (shows sections, bullets, and tags for local context managed by ByteRover CLI)

## Agentic Context Engineering command

### `br ace complete`

**Description:** Complete ACE workflow: save executor output, generate reflection, and update playbook in one command
